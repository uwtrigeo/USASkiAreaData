USA Ski Resorts

Summary of data contained on the maps

Washington State Avalanche Prone Areas
The avalanche prone areas in the boundaries of the six major Washington state ski resort are displayed. The data on this map was created by obtaining digital terrain models (DTM) for each ski area. The DTM for each resort was then analyzed in ArcGIS Pro where the slope and aspect was established. Those two layers were then combined and the North and East facing slopes with a pitch between 30 and 40 degrees were then isolated and applied to this map.

Size, Elevation, and Vertical Drop
The ski resort size, summit elevation, base elevation, and vertical drop for 365 resorts across the USA.

Snowfall, Lifts, Runs
The average snowfall from 2012-2022, number of lifts and runs.

Terrain
The percentage of the slope aspect within the ski resort boundary facing North, South, East, and West. And the difficulty of run type as a percentage of total runs at each resort.

Resort Scores
Each element displayed on the 'Size, Elevation and Vertical Drop', 'Snowfall, Lifts, Runs', and 'Terrain' and terrain maps was given a z-score to establish where each observed value is above or below the mean value, or average, in each element. The z-score, which typically occurs between -2 and +2, was then converted to a score between 1 and 20, 1 being for a z-score value below -1.8 and 20 for a value above +1.6. The sum of the converted z-score values in each element for each resort was used to provide a Total Score for a ski result with the higher scoring resorts representing a higher value above the mean in an element.

Resort Rankings
Each of the resort scores was then ranked to provide a number between 1 and 365 for each category to indicate where a resort placed within all the resorts. For the resort rankings 1 is the highest rank and 365 is the lowest.

State Stats
The totals number of resorts, acres, runs, lifts are displayed for each state.
